Maven Pipeline

This is a Java application that uses various DevOps tools and technologies to replicate a CI/CD pipeline. This application uses Docker, Groovy, Jenkins, Maven, Git, JUnit, Nexus and DigitalOcean. When the pipeline is triggered, the master branch will be checked out, tested, built into a JAR file, packaged into a docker-image and deployed to a private repository of Nexus on a remote-server.
